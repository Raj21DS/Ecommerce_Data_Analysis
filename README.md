# Ecommerce_Data_Analysis
Ecommerce_Data_Analysis

### Business Context
Use Spark DataFrames to inspect transactional data, join related tables, and generate simple business insights on products, regions, and customer activity.

### Objectives
 Read CSV files into Spark DataFrames and inspect schema, sample rows, and row counts.
 Clean essential columns and handle null values in join keys and descriptive fields.
 Apply beginner-level transformations such as selection, filtering, and new column creation.
 Join multiple tables to build a single analysis-ready DataFrame.
 Perform grouped aggregations and interpret simple business insights.
 Use cache/persist, repartitioning, and a broadcast join in an introductory way.
